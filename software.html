I've been working on learning how to write software since I first tried to develop ROMs for my slow ass android phone when I was a teen. This page is a great example that time spent doing something is not the only important metric and the quality of time spent practicing is just as important; as I only have the following packages that are worth sharing with others. All libraries are in rust unless specified otherwise.

  • (MHGL) Matt's HyperGraph Library - A library for working with undirected hypergraphs that can compute boundaries, links, stars, and skeletons. Working on adding homology algorithms and a simplex-tree datastructure.
  • SPARFF - Work in progress sparse linear algebra library over finite fields.
  • HDX_CODEC - Library developed to explore the error correcting codes developed in New Codes on High Dimensional Expanders and possible quantum counterparts.
